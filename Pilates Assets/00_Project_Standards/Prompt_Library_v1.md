# Prompt Library v1

## HERO
Použij referenční modelku PB40. Neměň místnost, kameru, světlo ani oblečení.
Změň pouze anatomicky správnou pozici těla.

## MASTER
Stejná pravidla jako HERO. Přidej prostor pro budoucí popisky.

## DETAIL
Stejná scéna. Zaměř se na čitelnost správného provedení cviku.

## OPRAVA
Nikdy neměň modelku ani prostředí. Oprav pouze požadovanou část anatomie.
