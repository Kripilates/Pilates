# Guide Card Standard

Official template: Guide Card Template v1.0
Reference file: `02_Exercise_Cards/Glute Bridge/glute_bridge_guide_card_v3.png`

Guide Card je navrzena primarne pro zobrazeni na telefonu.

Cil: po otevreni Guide Card musi uzivatel bez scrollovani videt:
- nazev cviku
- hlavni HERO fotografii
- prubeh START → HERO → START nebo START → HERO → END
- Dech
- Zamer se
- Opakovani

Guide Card musi byt citelna na beznem mobilnim displeji priblizne 390 × 844 px.

Pri navrhu preferuj:
- co nejvetsi hlavni fotografii
- co nejmene textu
- strucne popisky
- dostatek bileho prostoru
- zadne zbytecne graficke prvky

Step by Step slouzi jako podrobny navod. Guide Card nesmi suplovat jeho funkci.

Od Template v1.0 nemen:
- rozlozeni
- velikosti prvku
- barvy
- mezery
- typografii

U dalsich cviku se meni pouze:
- nazev cviku
- fotografie
- texty
- dech
- zamereni
- opakovani
